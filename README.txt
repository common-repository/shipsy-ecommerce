== Description ==
A WordPress Plugin to integrate functionalities provided by Shipsy to your WordPress website. This a generic plugin, which allows syncing, label download and tracking.

Features of Shipsy Econnect WordPress Plugin
● Syncing orders (zero-click, single or bulk)
● Downloading labels (single or bulk)
● Tracking of synced orders
● "Live" consignment status update
● Cancel order shipping

== Installation ==
● Make sure you have woocommerce plugin installed and activated successfully.
● In order to upload the plugin, navigate to Plugins -> Add New. Then click the blue Upload Plugin button and provide the route to your extension \'shipsy-econnect.zip\" archive and Click Install Now.  After successfully installing Activate the Plugin.
● On success, you should see a new item Shipsy Configuration in the menu items on the left side.
